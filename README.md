Descripción general del proyecto

Este proyecto aborda la predicción de salarios en ofertas de empleo mediante un flujo completo dividido en varias fases. En primer lugar, se realiza un proceso de web scraping para recopilar datos reales de ofertas de empleo relacionadas con el desarrollo de software. Posteriormente, los datos obtenidos se procesan y limpian utilizando técnicas de anál
